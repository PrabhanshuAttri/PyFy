# PyFy
Facebook message notifications on linux using Python without Graph API.

## Requirements

- Python 2.7+

- pip

  > sudo easy_install pip

- BeautifulSoup

  > pip install beautifulsoup4

## Usage
    export PYFY_FB_USER_EMAIL="your_email"
    export PYFY_FB_PASSWORD="your_fb_password"
    python PyFy.py
